# NumPy Basics

## Objective

To explore the fundamental features of the NumPy library used for numerical computing in Python.

## Description

This notebook demonstrates the basic operations of NumPy, including array creation, indexing, slicing, mathematical operations, aggregation functions, boolean masking, fancy indexing, reshaping, and structured arrays.

## Features Demonstrated

- Checking NumPy version
- Creating 0D, 1D, and 2D arrays
- Creating arrays of ones
- Indexing and slicing arrays
- Element-wise arithmetic operations
- Aggregation functions (Sum, Mean, Standard Deviation)
- Element-wise comparison
- Boolean masking
- Fancy indexing
- Array reshaping
- Structured arrays

## Technologies Used

- Python 3
- NumPy
- Jupyter Notebook

After completing this experiment, you will be able to:

- Filter data using boolean masking.
- Reshape arrays for different applications.
