# 📊 Student Marks Analysis

## 📌 Project Overview

This project analyzes student marks using **Python**, **Pandas**, and **Matplotlib**. It reads student data from a CSV file, calculates total and average marks, assigns grades, identifies the class topper, and generates visualizations.

---

## 🚀 Features

- Read student data from a CSV file
- Calculate total marks
- Calculate average marks
- Assign grades (A, B, C, D, F)
- Determine Pass/Fail status
- Find the class topper
- Display class statistics
- Generate data visualizations
- Save the analyzed report as a CSV file

---

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
