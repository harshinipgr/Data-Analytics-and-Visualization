# 💰 Sales Report Analysis

## 📌 Project Overview

This project analyzes sales data using Python, Pandas, and Matplotlib. It calculates revenue, identifies top-performing products, analyzes monthly sales, and generates visual reports.

---

## 🚀 Features

- Read sales data from a CSV file
- Calculate revenue for each order
- Compute total revenue
- Find highest revenue product
- Analyze monthly revenue
- Analyze category-wise revenue
- Generate charts
- Save processed sales report

---

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
